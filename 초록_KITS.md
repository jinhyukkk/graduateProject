# 기업 인사관리 시스템을 위한 LLM 기반 자기교정 Text-to-SQL 프레임워크

**An LLM-Based Self-Correcting Text-to-SQL Framework for Enterprise HR Management Systems**

**저자:** 박진혁  
**소속:** 국민대학교 대학원  
**대상 학회:** 한국IT서비스학회

## 국문 초록

기업 내부 데이터 활용이 확대되면서 SQL 비전문가의 데이터베이스 직접 조회 수요가 증가하고 있으나, 대형 언어 모델(LLM) 기반 Text-to-SQL 시스템은 여전히 높은 오류율을 보인다. 자기교정 요소를 포함한 기존 접근법(DIN-SQL, DAIL-SQL, CHESS 등)은 실행 오류에 대한 단순 재생성에 의존하여, 실행은 성공하지만 의미·논리적으로 잘못된 결과를 반환하는 오류 유형에는 효과적으로 대응하지 못한다. 본 연구는 Text-to-SQL에서 발생하는 오류를 구문·의미·논리의 3계층 10유형으로 정리한 새로운 오류 분류 체계를 제안하고, 오류 유형에 따라 교정 전략을 차별화하는 자기교정 프레임워크(SC-TSQL)를 설계한다. SC-TSQL의 핵심은 실행 기반 검증과 의미론적 일관성 검증(생성된 SQL을 자연어로 역번역한 뒤 원 질의와 NLI 기반으로 의도를 비교하는 방식)을 결합한 이중 검증 메커니즘으로, 감지된 오류 유형과 진단 결과를 교정 프롬프트에 주입하여 후속 교정의 정밀도를 높인다. 제안 프레임워크는 BIRD 벤치마크를 대상으로 GPT-4o를 동일 백본으로 사용하여 DIN-SQL, DAIL-SQL, CHESS 등 대표적 자기교정 베이스라인과 공정 비교 환경에서 평가되며, 어블레이션을 통해 의미 일관성 검증, 오류 유형 주입, 반복 교정 루프 각각이 성능에 기여하는 정도를 분석한다. 본 연구는 오류 유형 기반의 체계적 교정 전략이 실행 정확도와 의미적 정확성을 동시에 향상시킬 수 있음을 보이는 새로운 접근으로서, 향후 실제 기업 인사관리(HR) 도메인 데이터셋으로 확장하여 실 환경 적용 가능성을 검증할 계획이다.

**주제어:** Text-to-SQL, 자기교정, 대형 언어 모델, 오류 유형 분류, 기업 인사관리 시스템
